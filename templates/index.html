    🎯

    Objetivo

    Diseñar una red neuronal que prediga el riesgo de adicción y el bienestar mental a partir de datos conductuales y de estilo de vida.

    ⚙️

    Tecnología

    Red neuronal profunda Multi-Output con capas compartidas y dos cabezas especializadas, entrenada con TensorFlow / Keras.

    📊

    Datos

    Entrenada con más de 200,000 perfiles que incluyen hábitos de juego, salud física, interacción social y salud mental.

    💡

    Impacto

    Herramienta preventiva para detectar hábitos de riesgo, apoyo en investigación y base para aplicaciones de bienestar digital.
